Results - Display lists returned by wp query in widgets. Easly and customisable!

== Description ==

Display lists returned by wp query in easly customisable way. Lists customisations can be greatly extended by use of Admin Columns plugin. Just add another Results list widget, your own wp query parameters and place widget in the right place!

== Frequently Asked Questions ==

No questions yet.

== Installation ==

Upload the Results plugin to your blog, Activate it. Find it on widgets page.

1, 2, 3: You're done!
